SkillNova Bazaar deployment checklist

1. Frontend local development
   - Serve the frontend from `App/frontend` with:
     `python -m http.server 3000`
   - `App/frontend/config.js` already points to:
     `http://127.0.0.1:8000/api`

2. Backend local development
   - Start the backend on port 8000:
     `.\.venv\Scripts\python.exe -m uvicorn App.backend.app:app --host 127.0.0.1 --port 8000 --reload`
   - In `.env`, keep:
     `PORT=8000`
     `CORS_ALLOW_ORIGINS=http://127.0.0.1:3000,http://localhost:3000`

3. Frontend production change
   - Edit `App/frontend/config.js`
   - Change `apiBaseUrl` from:
     `http://127.0.0.1:8000/api`
   - To your Render backend URL, for example:
     `https://your-render-service.onrender.com/api`

4. Backend production CORS change
   - In Render environment variables, set:
     `CORS_ALLOW_ORIGINS=https://your-vercel-app.vercel.app`
   - If you want Vercel preview deployments to work too, also set:
     `CORS_ALLOW_ORIGIN_REGEX=https://.*\.vercel\.app`
   - If you use a custom frontend domain, add it to `CORS_ALLOW_ORIGINS` as a comma-separated value.

5. Backend production host and port
   - Render usually provides `PORT` automatically.
   - Do not hardcode `HOST` to a public domain.
   - Keeping `HOST=127.0.0.1` locally is fine.

6. Files involved
   - Frontend API base URL:
     `App/frontend/config.js`
   - Backend CORS config:
     `.env` or Render environment variables
   - Backend CORS parsing code:
     `App/backend/config/settings.py`
   - Backend CORS middleware:
     `App/backend/app.py`
